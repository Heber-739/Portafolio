Soy estudiante en programación desde Noviembre del 2021. Antes de eso, me formé en matemáticas en el profesorado de la Universidad Nacional de Misiones.

Desde entonces, eh estudiado de forma auto-didácta con recursos que pude encontrar en distintas páginas web como: FreeCodeCamp, Pildoras Informaticas, Soy Dalto, Udemy.

Actualmente me encuentro desarrollando 3 programas de formacion en programación:

One - Oracle Next-Education, en el que desarrollaremos habilidades Front-end (html, css, javascript), estudiaremos en profundidad el lenguaje Java y además brindan capacitaciones en soft skills. Finalización: Julio 2022.

Yo Quiero Programar de ConoSur Tech, en el que vamos a desarrollar el siguiente stack que consta de 6 diplomados: 1- HTML & CSS nivel básico; 2- Fundamentos de Programación; 3- JavaScript nivel básico; 4- Programación básica en C#; 5- Fundamentos de Bases de Datos; 6- Fundamentos de DevOps. Sin fecha estimada de finalización.

#YoProgramo es la segunda etapa del programa Argentina Programa. Se desarrolla en 9 modulos: 1- Introducción a desarrollo web y aplicaciones; 2- Front End estático (html, css y javascript); 3- Front End - Desarrollo Web Dinámico (Angular); 4- Bases de datos; 5- POO - Buenas prácticas de programación; 6- Backend - Java EE parte I; 7- Backend - Java EE parte II; 8- Backend - Java Spring boot; 9- Dev Ops. Finalización: Septiembre 2022.
